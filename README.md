# Personal Portfolio Website

This is a simple, modern and fully responsive personal portfolio website built only with **HTML** and **CSS**.  
The project is created as a learning exercise to practice front-end fundamentals and improve layout, design, and website structure skills.

---

## Features

- Modern and clean UI design  
- Fully responsive for desktop, tablet and mobile  
- Sticky navigation header  
- Hero section with introduction  
- About section  
- Skills section (using cards)  
- Projects section (card layout)  
- Experience section (professional tables)  
- Contact form (HTML only)  
- Custom CSS animations (fade-in, hover effects, gradients)  

---

## Technologies Used

- **HTML5**
- **CSS3 (Flexbox, Grid, Animations)**
- No frameworks and no JavaScript used.

---

## Purpose of This Project

This project is created for:

- Learning HTML and CSS deeply  
- Practicing layout techniques  
- Building a portfolio that can be used for job applications  
- Uploading a real project to GitHub for showcasing skills
